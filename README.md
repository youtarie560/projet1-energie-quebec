Contexte
Hydro-Québec publie des données ouvertes sur la consommation électrique de clients participant à un programme de gestion de la demande. 
Ces données incluent la consommation horaire, les conditions météorologiques, et des indicateurs d’événements de pointe.

## Setup
```git clone <link>.git``` <br>
```cd <folder-name>``` <br>
```cd hydro-ml-project``` <br>
```bundle exec jekyll serve```
